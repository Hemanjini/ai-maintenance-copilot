# AI Maintenance Copilot

An AI-powered HVAC monitoring system designed to reduce alert fatigue and help maintenance technicians identify critical issues faster.

## Problem

Traditional threshold-based HVAC alert systems generate excessive false alarms, causing technicians to lose trust in alerts and miss real failures.

## Goal

Build an intelligent, explainable maintenance assistant that:
- Detects meaningful anomalies
- Reduces noisy alerts
- Groups related incidents
- Explains why an alert matters
- Helps technicians prioritize action
